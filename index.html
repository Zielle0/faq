Popular Questions

What does the plugin do? ▼
The plugin scans websites and downloadable files in real-time to detect potential malware or harmful content using a deep-learning based detection system.
How does it detect malware? ▼
It uses a trained One Dimensional Convolutional Neural Network (1D-CNN) model that analyzes patterns in websites and files to determine whether they are safe or malicious.
Does it block downloads or websites automatically? ▼
No. The plugin alerts you if a file or site appears suspicious. You can then choose to proceed or cancel the action based on the risk warning.
What type of files can the plugin scan? ▼
Currently, it focuses on executable files (like .exe, .dll) and files with Portable Executable (PE) headers, which are common targets for malware.
Will this plugin slow down my browsing? ▼
No. The plugin is optimized to work in the background with minimal performance impact, scanning only when necessary.
Is my data sent to any server? ▼
No personal data is collected. Only essential file or URL metadata is temporarily processed for malware analysis and is not stored.
What should I do if a safe file is flagged as malicious? ▼
This is called a false positive. You can report it through the feedback form, and it will help improve future model training.
Is this plugin free to use? ▼
Yes, the plugin is completely free for personal use.
What if the scanning does not start automatically? ▼
You can manually start the scan by clicking the Scan button.
